# La Tabla - Gourmet Experience

Landing page for a premium charcuterie and gourmet catering service.

## Features

- Artisan charcuterie boards (Salamanca, Malasaña, Andalucía, Serrano)
- Grazing tables for events
- Chef specialties menu
- Classes and workshops
- Contact form for quotes

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS v4
- TypeScript
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```
